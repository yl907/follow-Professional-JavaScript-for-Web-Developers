# follow-Professional-JavaScript-for-Web-Developers
I read most part of 'Professional JavaScript for Web Developers' a few months ago, and now I can hardly remember what is it talking about. I think I need some practices to remember, so I record it, there will be some practices and guide, which can help people learning JavaScript concept.

具体做了什么：<br />
1. 类与对象: 对应《JavaScript高级程序设计》书中的第6章内容。 <br />
1.1. script/1_1object.js: 工厂模式，构造函数模式，原型模式。 <br />
1.2. script/1_2object.js: 原型链基础，以及一些类相关的函数。 <br />
1.3. script/1_3object.js: 介绍了类的一些构造方式。组合使用构造函数模式和原型模式(常用的类构造方式)，寄生构造函数模式。 <br />
1.4. script/1_4object.js: 介绍了如何构建原型链，如何使用借用构造函数。并将二者进行了组合，形成了一个实用的继承方法。<br />

2. 函数表达式: 对应《JavaScript高级程序设计》书中的第7章内容。 <br />
2.1. script/2_1function.js: 一个递归例子与在递归中如何使用arguments.callee的例子，简单介绍了闭包。 <br />

what have I done: <br />
1. class and object: Corresponding to Chapter 6 of the book "Professional JavaScript for Web Developers". <br />
1.1. script/1_1object.js: factory mode, constructor mode, prototype mode <br />
1.2. script/1_2object.js: basic prototype chain concept, and some functions relate to class. <br />
1.3. script/1_3object.js: explain some ways to build a class. combine constructor mode and prototype mode, parasitic constructor mode. <br />
1.4. script/1_4object.js: introduce how to build prototype chain, and how to use constructor stealing. And combine both, which is called combination inheritance, it is practical.<br />


2. function expression <br />
2.1. script/2_1function.js: A example of recursion, and how to use 'arguments.callee' in recursion. Briefly introduced the closure. <br />
